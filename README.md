# GroupMonitor
酷Q QQ机器人插件---群关键词监控  

    ###解决方案中包含四个项目 
    
    cn.xiaoxiaoge.GroupMonitor  
    插件项目  
     
    configini  
    配置文件的读写    我没有直接引用此项目的头，而是将此项目的cpp和头文件拷贝到其他项目中  
    
    GroupMonitorWin  
    使用MFC编写的dll界面  
    
    test  
    界面测试项目 此项目可以直接打开界面 方便单步调试  
    
详细功能请参考： [群关键词监控(VC)](https://cqp.cc/t/40793)
